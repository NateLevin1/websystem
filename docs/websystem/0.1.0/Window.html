forceClose(fadeopt)

Force the window to close. Note that this will not send a 'window-destroy' event, and the window will be destroyed without checking if it should be (e.g. checking if work has been saved) If sending a 'window-destroy' event is wanted, use the close() method instead.

Name Type Attributes Default Description

 true Whether or not to fade out when closing the window
