(static) addFileAtLocation(name, data, kind, parentPath, options)

Add a file with the parent parentPath.
Parameters:
Name Type Description
name String The name of the new folder.
data Blob | String The data of the new file. If string, the file will not be stored as binary.
kind String The kind of the data. E.g. "Image" or "App"
parentPath String The path to the parent of the new file.
options JSON The options to pass in.
